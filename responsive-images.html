Responsive Images in CSS

Width

If we use width 100% the image is able to grow larger than its original pixel dimensions. This means we will lose image quality.

Max Width

If we use max-width the image can shrink but it can't grow any larger than its original width so the quality is preserved

The Picture Element

The picture element allows us to set to different image sources. This might be helpful if we need to load a larger image for bigger screens
